# DreamCrafter 抽抽樂一覧

DreamCrafter 互動抽抽樂瀏覽

## 網址 Website URL

https://minixz.github.io/dreamcrafter-lottery-table/

## 功能 Features

- 像 Minecraft 一樣滑鼠懸停物品以顯示描述

  Hover items to show description like vanilla Minecraft
- 顯示抽抽樂鑰匙需要的錢或物品

  Show lottery keys required money or items
